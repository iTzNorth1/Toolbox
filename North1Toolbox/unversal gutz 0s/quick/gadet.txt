PART 2: USER GADGET INTERFACE
External Interface. Simple Letter-Based System. No Complexity.
════════════════════════════════════════════════════════════════════════════════

You have 7 gadgets. Pick one. Ask your question. Done.

════════════════════════════════════════════════════════════════════════════════
THE 7 GADGETS
════════════════════════════════════════════════════════════════════════════════

A: FINDER
What: Research something
Gets: Facts + sources + how confident + what's uncertain
Type: "A: What's the best way to learn?"

B: EXPLAINER
What: Understand how something works
Gets: Simple explanation + when it works + when it breaks
Type: "B: How does habit formation work?"

C: PLANNER
What: Get a step-by-step plan for something
Gets: Plan + timeline + resources + risks + how to undo
Type: "C: Create a fitness plan for 2 months"

D: VALIDATOR
What: Check if something is safe or smart
Gets: Assessment + strengths + risks + approval
Type: "D: Should I change jobs? [describe situation]"

E: THINKER
What: Need to think something through or are stuck
Gets: Questions + observations + new perspectives
Type: "E: I'm confused about two job offers"

F: ALL-IN-ONE
What: Want complete analysis (for big decisions)
Gets: Research + understand + plan + validate (all together)
Type: "F: Should I move cities?"

G: CUSTOM
What: You know exactly which gadgets you need
Gets: Just what you asked for
Type: "G: Run A + C" (research + plan, skip the rest)

════════════════════════════════════════════════════════════════════════════════
HOW TO USE
════════════════════════════════════════════════════════════════════════════════

Step 1: Pick a letter (A, B, C, D, E, F, or G)
Step 2: Type: [Letter]: [Your question/goal]
Step 3: System gives you answer
Step 4: Done
